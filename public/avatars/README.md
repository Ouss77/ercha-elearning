# Avatar Images

This folder contains predefined avatar images that users can select.

## Adding New Avatars

1. Add avatar images to this folder
2. Name them descriptively (e.g., `avatar-1.jpg`, `avatar-student.jpg`)
3. Recommended size: 200x200 pixels
4. Supported formats: JPG, PNG, WebP

## Default Avatars

- `default.png` - Default avatar when none is selected
